Avisos do projeto

O Companion Deck não fornece jogos, ROMs, ISOs ou BIOS. O usuário deve adicionar apenas arquivos que possui legalmente. O Code Deck é voltado a jogos offline, treino e formatos compatíveis com emuladores/cores. O app não injeta código, não altera memória de outros aplicativos, não burla anti-cheat e não oferece suporte a cheats online.
